# Morse-Notification-Tasker&Termux
A Tasker profile that will encode recieved notifications to morse code with Termux and vibrate the device with Termux API

## Instructions
* Make sure you have Tasker, Termux and Termux API installed on your device.
* Import the profile in Tasker by holding the Profiles section.
* Go to Tasker and select the app which you want to get the notifications from. (The default app selected within the profile is Whatsapp)
* Put "morse.sh" in ~/.termux/tasker/ on Termux.
